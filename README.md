# CCC IR Optimizer

This repository contains `my_opt.c`, a complete optimizer for intermediate representation (IR) code, designed for use with the CCC C front end translator, part of the CCC high-level synthesis tool.

## Features

- Dominator analysis
- Natural loop detection
- Region hierarchy
- Block-level constant propagation with partial evaluation
- Region-level constant propagation with fixpoint iteration
- Full dead-code elimination
- Copy propagation
- Loop-invariant code motion
- Strength reduction
- Common subexpression elimination
- Inline expansion
- Peephole optimization
- Control flow simplification
- Statistics: instruction count before/after optimization
- Support for TYPE_INT, TYPE_DOUBLE, TYPE_STRING
- Complete memory cleanup

## About

This optimizer was developed specifically for the CCC toolchain to improve IR code generated by the CCC C front end translator. It implements a wide range of classic and advanced optimization techniques to enhance code quality and performance.
